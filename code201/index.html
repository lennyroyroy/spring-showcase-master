Code 201

In our Intermediate Coding Class, our 201 students learned a lot about Web Development and how to build websites. The students used Glitch to host their websites, used Flexbox and CSS Grid, and repl.it, Code School, and Github!

Students were exposed to the wide world of Web Development tools to further their mastery. Some students built full applications while others created Gifs and built game on Code.Org. All students left the class with an interest in Computer Science officially sparked. Two students, Mannendri and Brios went above and beyond and created a template for our general SONYC Elective sites. Both were indispensable during the process of creating this site!
